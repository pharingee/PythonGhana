The Event Page on Python Ghana is to show current Python Related events happening around Ghana, You are also to register your event with the Python Event on the main python website, or you can let us know if we are to help you with that.

To have your event listed, Send a mail to events@pythonghana.org using this format:

Please add the following event:
                                
                                 * name of the event: 
                                 * type of event: 
                                 * focus on Python: 
                                 * approximate number of attendees: 
                                 * location (incl. region): 
                                 * dates/times/recurrence : 
                                 * link (in HTML format): 
                                        <a href="http://url/">name of the event</a>

Note: your events will also be forwarded to events@python.org

For recurring events, please also include the recurrence information, e.g. "monthly, every second Thursday".
